About Airline Management System

Airline Management System is a Software Engineering Project. Here User can signup and login to book flight tickets, print their boarding pass using the PNR, and they can also cancel the booked tickets. There is an admin panel, here admin can access the booked ticket details, he/she can add or delete flight schedules, add aircraft details, and activate or deactivate the aircraft. There is an about page and contact page also.
